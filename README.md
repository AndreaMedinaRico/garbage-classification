Este repositorio contiene el desarrollo de un modelo de clasificación de residuos en 6 tipos (vidrio, metal, cartón, papel, plástico y basura general).

Dentro de la carpeta raíz, se encuentran los siguientes archivos:

    <th>Descripción</th>
  </tr>
  <tr>
    <td>data_preparation_trash.ipynb</td>
    <td>Notebook que contiene toda la preparación de los datos y las iteraciones de modelado, así
        como la visualización de resultados.</td>
  </tr>
  <tr>
    <td>predict_pipeline.ipynb</td>
    <td>Notebook para realizar predicciones utilizando el modelo final entrenado. Requiere especificar

    </td>
  </tr>
  <tr>
    <td>README.md</td>
    <td>Archivo con la descripción general del proyecto.</td>
  </tr>

    <td>model_final.keras</td>

  </tr>
